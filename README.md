📖 About

MaintainIQ is a modern full-stack web application designed to simplify asset maintenance management. It allows organizations to manage assets, report maintenance issues through QR codes, assign technicians, monitor maintenance progress, communicate in real time, and receive AI-assisted issue analysis.

🚀 How to Use
👤 Admin
Login using Admin credentials.
Add, edit, and manage assets.
Generate QR codes for every asset.
View reported issues.
Assign technicians.
Track maintenance progress.
View dashboard statistics.
Chat with technicians in real time.
Monitor complete maintenance history.
👨‍🔧 Technician
Login using Technician credentials.
View assigned maintenance tasks.
Update issue status.
Add maintenance notes.
Chat with Admin.
Complete assigned maintenance jobs.
📱 Public User
Scan the QR code attached to an asset.
View asset information.
Report a maintenance issue.
Submit issue details for Admin review.
