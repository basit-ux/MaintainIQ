# 🚀 MaintainIQ — AI-Powered Asset & Issue Maintenance Management

MaintainIQ is a full-stack web application designed to simplify asset maintenance management. It enables organizations to manage assets, report maintenance issues using QR codes, assign technicians, monitor maintenance progress, communicate in real-time, and receive AI-assisted issue analysis.

# 📖 How to Use

## 👤 Admin

- Login using Admin credentials.
- Add, edit, and delete assets.
- Generate QR codes for assets.
- View all reported issues.
- Assign technicians.
- Monitor maintenance progress.
- View dashboard analytics.
- Chat with technicians in real-time.
- Access complete maintenance history.

---

## 👨‍🔧 Technician

- Login using Technician credentials.
- View assigned maintenance tasks.
- Update issue status.
- Add maintenance notes.
- Chat with Admin.
- Complete assigned maintenance jobs.

---

## 📱 Public User

- Scan an asset QR Code.
- View asset details.
- Report maintenance issues.
- Submit issue information directly to the Admin.
